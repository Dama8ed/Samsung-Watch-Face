# Samsung-Watch-Face
Here are different project for Samsung Watch Face Studio. You can import them into Watch Face Studio and modify.

[Watch Face Studio](https://developer.samsung.com/watch-face-studio/overview.html) official Samsung website.

## Instruction for import and set up on Samsung Galaxy Watch
1. Download [Watch Face Studio](https://developer.samsung.com/watch-face-studio/overview.html).
2. Open project.
3. Enable developer mode on the watch by clicking "software information" > "soft version" multiple times.
4. Enable debugging on the watch (ADB + wireless debug).
5. Connect watch and program by clicking "Run on device".
6. Fill up everything. **WARNING!** ports are different, check them on the watch.

[Instruction](https://developer.samsung.com/watch-face-studio/user-guide/test.html) from Samsung (not very useful).
